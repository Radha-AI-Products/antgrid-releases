# Antgrid — Releases

Desktop builds of [Antgrid](https://antgrid.ai), the agent-first IDE and command centre for AI coding agents (Claude Code, Codex, and more), with end-to-end-encrypted remote control from your phone.

This repository hosts downloads only — the source lives in a private repository.
